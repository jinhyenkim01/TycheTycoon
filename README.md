# 티케타이쿤 - TycheTycoon
### Java + JavaFX Digital Game
우아한테크코스 오픈미션 프로젝트
작성자: 김진현

## 소개

_당신의 기도가 드디어 빛을 발합니다!
운명의 변덕 자체를 다스리는 그리스 신화의 행운의 여신, 티케가 만신전에서 당신 앞에 현신했습니다.
그녀의 초월적인 행운이 초래하는 기적들은 과거 수많은 왕국들을 멸망으로부터 구원할 정도로 강력하죠...
당연히 오늘날 당신이 할 일은 정해져 있습니다. 카지노에서 대박을 터트리고 평생 떵떵거리며 삽시다! 자본주의여!_

_Your prayers have been answered!
From the Pantheons of Greek Mythology descends Tyche, the Goddess of Fortune who governs the whims of fate itself.
In the past, she was revered for her miracles that defies the most unlikeliest of odds that could save an entire civilization from the brink of ruin...
So naturally, you decide to abuse her power by hitting the Casino and getting filthy rich. Capitalism!_

## 프로젝트 설명

우아한테크코스 8기 프리코스에서 진행한 과제 (자동차 경주, 로또) 등 운적 요소가 있는 게임들을 소개하는 카지노에서 플레이하는 게임입니다.

전략적 요소를 추가하기 위해 플레이어는 본인의 "운"을 긍정적으로 강화할 수 있습니다. 처음에는 카지노에서 돈을 높은 확률로 잃게 되지만,
시간이 지날수록 강화된 운을 사용해 상금을 버는 게임입니다.

## 설치 방법

본 프로젝트는 JavaFX 21.0.6 및 BellSoft Liberica JDK 21.0.8 버젼을 토대로 작성되었습니다.

1. 리포지토리를 본인 환경에 클론합니다.
```
git clone https://github.com/jinhyenkim01/TycheTycoon.git
```
2. pom.xml 파일이 있는 저장소로 디렉토리를 설정한 후, Maven을 사용해 설치합니다.
```
mvn install
```
3. 다음 커맨드를 사용해 프로그램을 실행합니다.
```
java -jar target/tyche_tycoon-1.0-SNAPSHOT-shaded.jar
```
